简介

CSS(Cascading Style Sheets,层叠样式表)。样式表定义如何显示 HTML 元素，就像 HTML 中的字体标签和颜色属性所起的作用那样。样式通常保存在外部的 .css 文件中。我们只需要编辑一个简单的 CSS 文档就可以改变所有页面的布局和外观。

使用例子

Text

  • 颜色

    • color
      • rgb(x,y,z)
      • name(‘red’,‘blue’)
      • 十六进制 #ff0000

  • 对齐方式

    • text-align
      • center
      • right
      • justify

  • 文字修饰

    • text-decoration
      • none (去掉修饰，可用于网址的下划线)
      • overline
      • line-through
      • underline

  • 文本转换

    • text-transform
      • uppercase(转化为大写)
      • lowercase(转化为小写)
      • capitalize(首字母大写)

  • 文本缩进

    • text-indent

Front

  • CSS字形
    • 安全字体
  • 字体系列 **注意：**当字体名字在英文状态下需要空格分开、中文情况下大于等于两个字时，需要用引号""
    • 当浏览器不支持第一种字体时，将自动寻求下一种
  • 字体样式
    • font-style
      • normal
      • italic
      • oblique
  • 字体大小
    • front-size
    • 绝对大小
      • px
    • 相对大小
      • em (默认为16px)
      • rem
